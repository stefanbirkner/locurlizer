Locurlizer

A servlet filter for multilingual URLs. You create your application with human readable URLs and provide a translation file of these URLs. Locurlizer translates the URLs on the fly without changing your application.

Overview from 10.000 ft


Example How it works

You have this

http://yourdomain.com/help.jsp

…
<filter>
   <filter-name>Locurlizer</filter-name>
   <filter-class>Locurlizer</filter-class>
   <init-param>
      <param-name>configuration</param-name>
      <param-value>locurlizer.json</param-value>
   </init-param>
</filter>
…

locurlizer.json

{
  "default-language":"en",
  "language-resolver":"com.github.stefanbirkner.locurlizer.TopLevelDomainLanguageResolver",
  pathes:[
    {
      en:"/help.jsp",
      de:"/hilfe.jsp"
    },
    {
      en:"/contact.jsp",
      de:"/kontakt.jsp"
    }
  ]
}

help.jsp

…
<html>
  <head>
    <title><fmt:message key="title"/></title>
  </head>
  <body>
    <p>
      <fmt:message key="text"/>
      <a href="<c:url value="/contact.jsp"/>"></a>
    </p>
  </body>
</html>

You get this

http://yourdomain.com/help.jsp

<html>
  <head>
    <title>English Title</title>
  </head>
  <body>
    <p>
      English text
      <a href="/contact.jsp"></a>
    </p>
  </body>
</html>

http://yourdomain.de/hilfe.jsp

<html>
  <head>
    <title>Deutscher Titel</title>
  </head>
  <body>
    <p>
      Deutscher Text
      <a href="/kontakt.jsp"></a>
    </p>
  </body>
</html>
